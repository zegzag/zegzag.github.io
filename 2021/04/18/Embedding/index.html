Embedding

发表于 2021-04-18 | 更新于: 2021-04-18 | 阅读数

word2vec

  • 基本方法   设词表大小为\(V\)，则每个样本(词)表示为\(x_i=(0,...,1_{i},...0)\)   两层神经网络: 从输入层到隐层: \(A\in R^{V\times K}\)，从隐层到输出层: \(B\in R^{K\times V}\)， 表示为embedding向量即为 \(A=(\mathbf{a_1}, \mathbf{a_2}, ..., \mathbf{a_V})^T, B=(\mathbf{b_1^T}, \mathbf{b_2^T}, ..., \mathbf{b_V^T})\)， 针对某个输入词\(w_I\)，令\(u_j = \mathbf{a}_I\mathbf{b}_j^{T}\)， 那么我们从输入词 \(w_I\) 到输出词 \(w_O\) 之间的条件概率\(P(w_O|w_I)\)为: \[ \hat{y_O} = P(w_O|w_I) = \frac{exp(\mathbf{a}_I\mathbf{b}_O^T)}{\sum_{j=1}^{V}exp(\mathbf{a}_I\mathbf{b}_j^T)}=\frac{exp(u_O)}{\sum_{j=1}^{V}exp(u_j)} \tag{1} \]   根据极大似然概率我们有Loss函数： \[ L = -log(P(w_O|w_I)) = -log(y_O)=log(\sum_{j^=1}^{V}exp(u_j))-u_O \tag{2} \]   那么样本空间 \(\mathcal{D}\) 上的极大似然(Loss函数)为： \[ \mathcal{L} = -\sum_{(w_O, w_I)\in \mathcal{D}}log(P(w_O|w_I)) \tag{3} \] 对输出隐向量求偏导，针对每一项\(u_j\)(每一个输出隐向量)的偏导为，这里的\(e_j\)可以看做是一种误差: \[ \frac{\partial L}{\partial u_j}=\frac{exp(u_j)}{\sum_{j^{'}=1}^{V}exp(u_{j^{'}})}-\mathbb{I}_{j=O}=y_j-t_j:=e_j \tag{4} \] \[ \frac{\partial L}{\partial \mathbf{b_j}}=\frac{\partial L}{\partial u_j}\frac{\partial u_j}{\partial \mathbf{b_j}}=e_j*\mathbf{a_i} \tag{5} \] 参数更新，可以看到针对一对输入输出词对$ {w_I, w_O}$，都要更新全部的输出隐向量，计算量非常大： \[ \mathbf{b_j}^{new} \leftarrow \mathbf{b_j}^{old}-\eta*e_j*\mathbf{a_i}\quad for \ j \ in \ \{1, ..., V\} \tag{6} \]

    • 加入\(b_j\), \(a_j\)都是正向量时，当\(j==O\)时，\(e_j < 0\)，此时\(b_j\)各分量增大，当\(j!=O\)时，\(e_j > 0\), 此时\(b_j\)个分量减少。

    对输入隐向量求偏导，由链式法则得： \[ \frac{\partial L(u_1, ..., u_V)}{\partial \mathbf{a}_I} = \sum_{j=1}^V\frac{\partial L}{\partial u_j}\frac{\partial u_j}{\partial \mathbf{a}_I} =\sum_{j=1}^{V}e_j\mathbf{b}_j \tag{7} \] 参数更新： \[\mathbf{a}_I^{new} \leftarrow \mathbf{a}_I^{old}-\eta*\sum_{j=1}^Ve_j\mathbf{b_j} \tag{8}\]

  • 模型变种-CBOW模型
      由多个词的上下文来预测一个词 \(P(w_O|w_I) \rightarrow P(w_O|w_{I_1}, w_{I_2}, ..., w_{I_C})\)， 此时\(u_j=\frac{1}{C}(\mathbf{a}_{I_1}+...+\mathbf{a}_{I_C})\mathbf{b}_j^T\)， 剩下的参数更新就不写了。

  • 模型变种-SkipGram模型
      一个词预测多个上下文\(P(w_O|w_I) \rightarrow P(w_{O_1},...,w_{O_C}|w_I)=\prod_{o=1}^CP(w_o|w_I)\)， 剩下的Loss函数和参数更新就不写了。

  • 复杂度分析
      CBOW模型针对每个样本组合\((w_{I_1}, ..., w_{I_C}, w_O)\)需要更新\(O(V)\)个输出层的隐向量，Skip-Gram为\(C*V\)个， 而每个隐向量的计算涉及到\(softmax\)，复杂度为\(O(V)\)，总的复杂度为\(O(V^2)\)

  • 加速训练-Hierarchical Softmax
      分层Softmax的主要思想是降低\(softmax\)的复杂度\(O(V) \rightarrow O(log(V))\)。将词表\(\mathbb{V}\)构建成一棵二叉树(通常为哈夫曼树)，该树共有\(V\)个叶子节点，则有\(V-1\)个中间节点。

    • 定义：

      • \(n(w, j)\)为从\(root\)到单词\(w\)路径上的第\(j\)个节点，则\((w, 1)=root\)
      • \(L(w)\)为从\(root\)到单词\(w\)上的路径长度，则\(n(w, L(w))=w\)
      • 对于任意一个中间节点\(\mathbf{v^{'}}_t\)，给定输入向量\(w_I\)后，从该节点往左走的概率为\(\sigma(\mathbf{v}^{'}_t\mathbf{v}_{w_I})\)， 那么往右走的概率为\(\sigma(-\mathbf{v}^{'}_t\mathbf{v}_{w_I})\)

    • 概率分布，由以上定义可知： \[ P(w_O|w_I)=\prod_{j=1}^{L(w_O)-1}\sigma(sig(n(w,j+1)==lc(n(w,j)))\mathbf{v}_{n(w,j)}^{'}\mathbf{v}_{w_I}^T) \]

      • 其中 \[ sig(x)=\begin{cases} 1 \quad x\ is \ true \\ -1 \quad x\ is \ false \end{cases} \]
      • \(\sigma(x)=\frac{1}{1+e^{-x}}\)
      • \(lc(n):= leftchild \ \ node\ \ of\ \ n\)
      • 可以证明\(\sum_{O=1}^VP(w_O|w_I)=1\)

    • 由概率分布的Loss函数如下： \[L =-log(P(w_O|w_I)) =-\sum_{j=1}^{L(w_O)-1}log(\sigma(sig(n(w,j+1)==lc(n(w,j)))\mathbf{v}_{n(w,j)}^{'}\mathbf{v}_{w_I}^{T}) \]

  • 加速训练-Negative Sampling
      传统的参数更新针对每一个\((w_O, w_I)\)对都要更新全部词表的emb向量(softmax中分母)，负采样的思想是从全部词表中随机采样一批负向单元，仅仅利用这批负向单元来更新。   令\(\mathcal{W}_{neg}=\{j_{neg_1}, j_{neg_2}, ..., j_{neg_K}\}\)，对于真实的输出分布，我们有： \[ \hat{y_j} = P(w_j|w_I)=\frac{exp(u_j)}{\sum_{j^{'}=1}^Vexp(u_{j^{'}})} \] 那么在负采样的情况下，我们有： \[ \hat{y_j}=P(w_j|w_I)=\begin{cases} \frac{exp(u_j)}{\sum_{j^{'}\in \mathcal{W}_{neg} \cup \{O\}}exp(u_{j^{'}})}, \quad if\ j \in \mathcal{W}_{neg}\cup\{O\}\\ 0, \quad else \end{cases} \] 论文直接从对数损失的角度给出了Loss函数： \[ L = -log(\sigma(\mathbf{a}_I\mathbf{b}_O^T)) - \sum_{j\in \mathcal{W}_{neg}}log(\sigma(-\mathbf{a}_I\mathbf{b}_O^T)) \]
